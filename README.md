# Happy App 🌞

An app to help track 📝 and increase 📈 happiness 🌞

## Dependencies

Ruby v2.6.0  
Rails v6.0.2
PostgreSQL

[Bootstrap Ruby](https://github.com/twbs/bootstrap-rubygem/blob/master/README.md)

## How to use

```sh
# install dependencies
bundle install
# run the app on localhost:3000
rails serve
```

## Databases

```sh
# To create databases
rake db:create
```
